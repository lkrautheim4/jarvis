WINDOWS AUTO-STARTUP
To make Jarvis open automatically when Windows starts:
1. Press Win + R → type shell:startup → Enter
2. Copy a shortcut of JARVIS.html into that folder
3. Right-click shortcut → Properties → set "Open with" to Chrome
4. Done — Jarvis opens first on every boot
